# Project Readmes

This folder contains all generated README documentation for Python modules in the repository.

Structure
- `pages/` — README files for page object and base page modules.
- `api/` — README files for API helper modules.
- `utils/` — README files for utility modules.
- `tests/api/` — README files for API tests.
- `tests/web/` — README files for web UI tests.
- `root/` — README files for top-level project modules such as `conftest.py`.

How to use
- Open the matching `.md` file under `readmes/` to review the detailed explanation for a module.
- Each file is grouped by module type and includes call mappings for test files.

Example
- `readmes/pages/base_page.md` explains the Playwright `BasePage` helper.
- `readmes/tests/web/test_login_page.md` explains the UI login test and the page objects it calls.
